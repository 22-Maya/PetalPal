# PetalPal

PetalPal is a smart plant pot system supported by a companion app. It helps new gardeners learn how to care for their plants and gives busy users an easier way to manage watering schedules and plant maintenance.

## Features

* Smart pot and app integration for guided plant care
* Beginner friendly plant explanations and tips
* Reminders and simple tracking for busy users
* Clean, modular structure for future expansion
* Built with Swift for a smooth iOS experience
